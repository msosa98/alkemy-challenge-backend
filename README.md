## Usage

Import the database.sql file into your database manager
```
./docs/sql/database.sql
```

Preload the data from the data.sql file into your database manager
```
./docs/sql/data.sql
```


Clone the project and install the dependencies
```
git clone https://github.com/msosa98/alkemy-challenge-backend.git
cd alkemy-challenge-backend
npm install
```

Run the application
```
npm start
```
